🏥 Healthcare Provider Analytics Dashboard
📌 Overview

An interactive Power BI dashboard analyzing hospital billing and operational data to uncover revenue trends, department performance, and cost distribution.

Designed to provide financial insights across procedures, service types, insurance coverage, and patient payments.

📊 Key Highlights

Total Billing, Treatment & Medication Cost Analysis

Department-wise Revenue Contribution

Procedure-level Billing Breakdown

Diagnosis vs Service Type (Emergency / Inpatient / Outpatient)

Insurance vs Out-of-Pocket Comparison

Dynamic Filters (Patient Race, Year, Quarter, Month)

🛠 Tools Used

Power BI | DAX | Data Modeling | Data Cleaning | Interactive Visualizations
